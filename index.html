Quick Start

To get started with any of these projects locally, clone the relevant GitHub repository and follow the setup instructions:

bash 
git clone https://github.com/nourijp/portfolio-hub.git
cd portfolio-hub
npm install && npm run dev

Infrastructure

All sites are deployed using a combination of GitHub Pages for static hosting and Cloudflare for DNS, CDN, and HTTPS. Uptime is monitored via UptimeRobot and reported on the hub dashboard.

💡
All subdomains use Cloudflare proxying for automatic HTTPS, DDoS protection, and edge caching.

  1. DNS via Cloudflare

    Each subdomain has a CNAME record pointing to nourijp.github.io, with Cloudflare acting as a proxy.

  2. GitHub Pages

    Each repository has a CNAME file matching its custom domain, and a GitHub Actions workflow that builds and deploys on push to main.

  3. Uptime Monitoring

    UptimeRobot checks each endpoint every 5 minutes and reports live/down status to the hub dashboard via a status page API.
